모두의 가계부 개인정보처리방침

시행일: 2025년 1월 1일

1. 개인정보 수집 및 이용 안내

모두의 가계부는 사용자의 개인정보를 일체 수집하지 않습니다.

본 앱은 완전한 로컬 기반 애플리케이션으로, 모든 데이터는 사용자의 기기에만 저장되며 외부 서버로 전송되지 않습니다.

2. 데이터 저장 및 보안

2.1 로컬 저장

  • 모든 가계부 데이터(예산, 지출 내역 등)는 사용자의 기기에만 저장됩니다
  • iOS의 SwiftData를 활용하여 안전하게 로컬 저장소에 보관됩니다
  • 네트워크 연결 없이도 모든 기능을 사용할 수 있습니다

2.2 데이터 보안

  • 사용자 데이터는 iOS의 기본 보안 체계에 의해 보호됩니다
  • 기기의 암호화 설정에 따라 데이터가 암호화되어 저장됩니다
  • 앱 삭제 시 모든 데이터가 기기에서 완전히 제거됩니다

3. 제3자 제공 및 공유

모두의 가계부는 어떠한 개인정보도 수집하지 않으므로, 제3자에게 제공하거나 공유할 정보가 없습니다.

  • 광고 네트워크 연동 없음
  • 분석 도구 사용 없음
  • 소셜 미디어 연동 없음
  • 클라우드 백업 기능 없음

4. 사용자 권리

4.1 데이터 관리

  • 사용자는 앱 내에서 언제든지 자신의 데이터를 확인, 수정, 삭제할 수 있습니다
  • 앱을 삭제하면 모든 데이터가 즉시 기기에서 제거됩니다
  • 기기 변경 시 iOS의 기본 백업/복원 기능을 통해서만 데이터 이전이 가능합니다

4.2 데이터 내보내기

향후 업데이트를 통해 CSV 등의 형식으로 데이터를 내보내는 기능이 추가될 수 있습니다. 이 경우에도 사용자가 직접 선택한 경로로만 저장되며, 외부 서버로는 전송되지 않습니다.

5. 앱 권한

모두의 가계부는 다음과 같은 최소한의 권한만을 요청합니다:

  • 알림 권한 (선택): 예산 리마인더 기능 사용 시에만 필요하며, 거부해도 앱 사용에 지장이 없습니다

위치 정보, 카메라, 마이크, 연락처 등의 민감한 권한은 일체 요청하지 않습니다.

6. 아동의 개인정보 보호

본 앱은 개인정보를 수집하지 않으므로 아동의 개인정보 역시 수집되지 않습니다. 모든 연령대의 사용자가 안전하게 사용할 수 있습니다.

7. 개인정보처리방침 변경

본 개인정보처리방침이 변경될 경우, 앱 업데이트 노트를 통해 사전에 공지하겠습니다. 중요한 변경사항이 있을 경우 앱 내 공지사항을 통해 별도로 안내드립니다.

9. 기술적 세부사항

9.1 데이터 저장 방식

  • SwiftData 프레임워크를 사용한 로컬 데이터베이스
  • iCloud 동기화 비활성화
  • 외부 API 연동 없음

9.2 보안 조치

  • iOS 샌드박스 환경 내에서만 실행
  • 다른 앱과의 데이터 공유 차단
  • 네트워크 통신 기능 없음

요약: 모두의 가계부는 사용자의 프라이버시를 최우선으로 생각합니다. 어떠한 개인정보도 수집하지 않으며, 모든 데이터는 사용자의 기기에만 안전하게 보관됩니다.
